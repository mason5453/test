开发 C#.NET 更简单？
是。.NET 提供海量现成类库（无需手写底层逻辑），开发效率提升数倍
做 Web 更简单？
是。.NET 内置 ASP.NET Core 框架，C# 纯语言无 Web 能力，必须依托 .NET
做 CMD（控制台）更简单？
是。Console 类是 .NET 提供的，纯 C# 无法输出 / 读取控制台内容
.NET 程序 Windows 移 Linux？
是。.NET Core/5+ 跨平台，编译后的程序可直接在 Linux 运行 （无特殊修改）
.NET 程序 Linux 移 Windows？
是。同上，跨平台双向兼容（仅需确保目标系统装了.NET 运行时）
.NET 有更多扩展？
是。.NET 生态有 NuGet 包管理器（数十万个现成扩展库），纯 C# 无扩展体系
以上纯 C# 都做不了？
是。纯 C# 仅语法，所有实际功能（IO、Web、跨平台）都依赖.NET
